# PlayStation 5 emulation

PS5 research, per-game sections, and descriptively named helpers belong directly
in `emulators/ps5/`. Local game files belong in
`$EMULATION_ROOT/roms/ps5/`, never in this repository.

This is reserved organizational space, not a working-support claim. The current
managed RetroArch setup has no PS5 core, and any standalone PS5 emulator must be
treated as a new experimental case with compatibility verified before publishing
instructions.
