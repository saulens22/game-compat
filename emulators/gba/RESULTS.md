# Pokemon Radical Red final results

The official Radical Red 4.1 UPS patch applies locally on Linux to an exact,
clean FireRed 1.0 USA source. The patcher validates the source, pinned
official patch archive, embedded UPS CRCs, and final output hash. The resulting
ROM launches through Steam RetroArch with the managed current mGBA core and
Steam Input. In-game controller input was confirmed with label-matched A/B,
X/Y as Select/Start, Menu opening RetroArch without also sending Start, separate
LB/RB shoulders, left-stick movement, LT rewind, and immediate capped 4x RT
fast-forward with audio retained.
