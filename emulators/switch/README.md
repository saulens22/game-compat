# Nintendo Switch emulation

Switch research, per-game sections, and descriptively named helpers belong
directly in `emulators/switch/`. Local game files belong in
`$EMULATION_ROOT/roms/switch/`, never in this repository.

This system is reserved for a standalone-emulator workflow. The current managed
RetroArch setup does not install or claim support for a Switch Libretro core.
Select and document a standalone emulator per case after legal dump format,
controller path, graphics backend, and game compatibility are verified.
