# Nintendo Switch emulation results

- A native Linux Ryujinx 1.3.3 installation recognized Pokémon Scarlet and
  booted Pokémon Violet through Vulkan on the NVIDIA GPU.
- The shared launcher can select a title and request fullscreen without forcing
  a resolution that may be wrong for another display.
- Docked 3× scaling was GPU-heavy and coincided with graphics-pipeline misses;
  2× scaling and a current game update still need testing.
- Steam shortcut integration is prepared but not yet confirmed.
